```ruby
a = [1, 2, 3, 5, 7, 9]
a = [1..3]              # => [3,5,7]
a = [1...3]             # => [3,5]
a = [3..3]              # => [7]
a = [-3..-1]            # => [5,7,9]
```

위 코드만 봐도 `..`과 `...` 차이를 알 수 있다.

`..`은 시작, 끝 인덱스의 객체 참조를 뽑아 새로운 배열을 만들어 반환한다.

<br>

`[start, count]` 처럼 숫자 쌍으로 지정할 수도 있다.

```ruby
a = [1,3,5,7,9]
a[1, 3]         # => [3,5,7]
```

`start`에서 `count`만큼 객체 참조를 뽑아서 새로운 배열을 만들어 반환한다.
